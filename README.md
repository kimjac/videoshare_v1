🎬 VideoShare

VideoShare is a lightweight and extensible video sharing platform built with PHP, MySQL, AJAX, and Bootstrap.

It is designed for developers and independent publishers who want full control over their video infrastructure without relying on closed platforms.

The system includes video management, user interaction, and engagement-based content ranking, all structured in a modular way to support future expansion.

🔹 Key Highlights
Video upload and thumbnail support
User system with admin roles
AJAX-based comment system with replies and voting
Engagement-driven content sorting
Responsive design with dark/light mode
Multi-language support (EN / DK)
SEO-ready (sitemap + robots.txt)

🔹 Status
Version 1 (Prototype)
Functional and ready for testing, with focus on iterative development.

🔹 Quick Start
1. Upload files to your server
2. Import database.sql into MySQL
3. Configure includes/config.php
4. Open in browser

🔹 Vision
VideoShare is built as a foundation for scalable media systems, including future support for AI-based metadata, content recognition, and network-driven traffic models.
